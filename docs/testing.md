# 测试

Rust 是一门非常重视正确性的语言，这门语言本身就提供了对编写软件测试的支持。

测试有三种风格：

* [单元][unit]测试。
* [文档][doc]测试。
* [集成][integration]测试。

Rust 也支持在测试中指定额外的依赖：

* [开发依赖][dev-dependencies]
